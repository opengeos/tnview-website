# Geospatial Data

Below is a list of geospatial data resources for Tennessee, the U.S., and global use.

## Tennessee Data Sources

| Name | Description |
| --- | --- |
| [TNMap Open Data Portal](http://tn-tnmap.opendata.arcgis.com) | The state’s new portal making available data including administrative boundaries, political districts, water, state parks, and trails. |
| [STS GIS Data](https://www.tn.gov/finance/sts-gis/gis/data/) | STS GIS Data Download state LiDAR and derived products including elevation (DEM), contours, and building footprints. |
| [TNGIS Tennessee GIS Clearinghouse](http://www.tngis.org/) | Provides links to other data sources and some Tennessee specific downloadable products derived from Federal datasets. |
| [TWRA GIS](https://www.tn.gov/twra/gis-maps.html) | Tennessee Wildlife Resources Agency maintained maps and data. |
| [Tennessee Property Viewer](http://tnmap.tn.gov/assessment/) | Interactive web map allowing users to browse property boundaries for the majority of Tennessee counties. No data download functionality. Some counties including Knox, Davidson, Shelby and Hamilton are not included. |
| [Tennessee GIS Data at UTK Libraries](https://libguides.utk.edu/tngis/data) | Collection of links to data sources organized by theme maintained by Eric Arnold, UTK Libraries GIS Specialist. |
| [Tennessee Spatial Data Nexus](http://opentn-myutk.opendata.arcgis.com/) | ArcGIS Open Data Site maintained by the UTK Geography GIS Labs |
| [Tennessee State Data Center](http://tndata-myutk.opendata.arcgis.com/) | ArcGIS Open Data site hosting census and population data maintained by the Tennessee State Data Center |
| [TN Geographic Alliance Open Data Portal](https://data-tga.opendata.arcgis.com/) | Ready to use global, national, and state data |
| [Opioid data](https://giswqs.github.io/Opioids-Tennessee) | Opioid data for Tennessee and the nation |
| [Tennessee and Landsat by USGS](https://pubs.er.usgs.gov/publication/fs20223063) | Landsat resources for Tennessee by the [National Land Imaging Program](https://www.usgs.gov/programs/national-land-imaging-program) |

## US Federal Government

| Name | Description |
| --- | --- |
| [US Census Bureau TIGER products](https://www.census.gov/geo/maps-data/data/tiger.html) | TIGER products are spatial extracts from the Census Bureau’s MAF/TIGER database, containing features such as roads, railroads, rivers, as well as legal and statistical geographic areas. |
| [Data.GOV](https://www.data.gov/) | Comprehensive index of US Government open data including spatial and non-spatial. |
| [USGS National Map Viewer](https://viewer.nationalmap.gov/basic/) | Elevation (DEM), hydrography, imagery, land cover and other US Geological Survey products are downloadable from this interactive map. [Go here for info](https://nationalmap.gov/) on this tool and available data. |
| [EPA Geospatial Resources](https://www.epa.gov/geospatial) | Applications and searchable datasets maintained by the US Environmental Protection Agency. |
| [NRCS Geospatial Gateway](https://datagateway.nrcs.usda.gov/) | US Department of Agriculture – Natural Resources Conservation Service resource includes soil, imagery, hydrology and climate data. |
| [NOAA Digital Coast](https://coast.noaa.gov/digitalcoast/) | Resource for a variety of coastal datasets and geospatial tools including extensive collection of elevation and LiDAR data. |
| [US Fish & Wildlife Service GIS Data](https://www.fws.gov/gis/data/national/) | Data related to wildlife conservation and fisheries including critical habitat areas, ecosystem regions, and the National Wetlands Inventory. |
| [National Park Service Data Clearinghouse](https://www.nps.gov/gis/data_info/) | Data for NPS administered areas including National Parks, Monuments, Historic Sites and Seashores. |
| [National Renewable Energy Laboratory GIS Data](https://www.nrel.gov/gis/data.html) | GIS resources relating to renewable energy including biomass, solar, geothermal and wind data. |
| [USGS Earth Resources Observation and Science Center](https://eros.usgs.gov/find-data) | Collection of satellite and aerial imagery and derived products including land cover and fire maps. |

## US Non-Government

| Name | Description |
| --- | --- |
| [NSGIC GIS Inventory](https://www.gisinventory.net/) | Multi-agency resource compiling datasets from state and local providers managed by the [National States Geographic Information Council](https://www.nsgic.org/) |
| [Texas A&M GeoCoder](https://geoservices.tamu.edu/Services/Geocode/) | Free geocoding engine for US addresses provided by Texas A&M University Geoservices. |
| [The Nature Conservancy’s Geospatial Conservation Atlas](https://geospatial.tnc.org/) | It includes web maps, apps, a map gallery, local initiative portals, conservation data, and other resources. |
| [Open Topography](http://www.opentopography.org/) | Download high resolution topographic data in raster or point cloud formats. Most datasets are within US but some global availability. |

## Global & International Resources

| Name | Description |
| --- | --- |
| [GeoData at Tufts](https://geodata.tufts.edu/) | Comprehensive portal containing over 13,000 open datasets. |
| [United Nations Environment Programme Data Explorer](http://geodata.grid.unep.ch/) | Its online database holds more than 500 different variables, as national, subregional, regional and global statistics or as geospatial data sets (maps), covering themes like Freshwater, Population, Forests, Emissions, Climate, Disasters, Health and GDP |
| [WorldClim](http://www.worldclim.org/) | Global Climate Data including rasters for temperature, precipitation, wind and solar radiation. |
| [ESRI Living Atlas of the World](https://livingatlas.arcgis.com/) | The foremost collection of geographic information from around the globe. It includes maps, apps, and data layers to support your work. |
| [Global Land Cover Facility](http://glcf.umd.edu/) | GLCF develops and distributes remotely sensed satellite data and products that explain land cover from the local to global scales |
| [HydroSHEDS](https://hydrosheds.cr.usgs.gov/index.php) | Global hydrographic and watershed data created by a collaboration of the USGS and World Wildlife Fund. |
| [FAO GeoNetwork](http://www.fao.org/geonetwork/) | Global resource for maps, GIS data and imagery maintained by the Food and Agriculture Organization of the United Nations. |
| [GADM Database of Global Administrative Areas](http://gadm.org/) | Administrative areas in this database are countries and lower level subdivisions such as provinces, departments, bibhag, bundeslander, daerah istimewa, fivondronana, krong, landsvæðun, opština, sous-préfectures, counties, and thana. |
